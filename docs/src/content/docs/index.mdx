## Features

<ChameleonCardGrid>
  <ChameleonCard title="Per-Domain Rules" icon="pencil">
    Create CSS rules that apply only to specific domains or globally across all sites.
  </ChameleonCard>
  <ChameleonCard title="Live Editing" icon="seti:css">
    Edit CSS with a full-featured CodeMirror editor with syntax highlighting and autocomplete.
  </ChameleonCard>
  <ChameleonCard title="Quick Toggle" icon="approve-check">
    Enable or disable stylesheets instantly from the popup without opening the full editor.
  </ChameleonCard>
  <ChameleonCard title="Palettes" icon="star">
    Group related styles into palettes and apply them with a single click.
  </ChameleonCard>
